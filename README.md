# Galaxy Classification using Machine Learning

## Overview

This project applies machine learning and computer vision techniques to classify galaxies based on their morphology. Using astronomical imaging data, the model learns to distinguish between different galaxy types such as spiral, elliptical, and irregular galaxies.

The project combines concepts from astrophysics, data science, image processing, and deep learning.

---

## Objectives

- Understand galaxy morphology
- Process astronomical image data
- Build image classification models
- Apply deep learning techniques to astrophysical datasets
- Evaluate model performance on real observations

---

## Scientific Background

Galaxies are broadly classified into different morphological categories:

### Spiral Galaxies
Characterized by rotating disks and spiral arms containing gas, dust, and young stars.

### Elliptical Galaxies
Smooth, nearly featureless systems dominated by older stellar populations.

### Irregular Galaxies
Galaxies that do not fit standard morphological classifications and often exhibit chaotic structures.

Galaxy classification plays an important role in understanding galaxy formation, evolution, and large-scale structure in the universe.
